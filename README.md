 The popular Tic Tac Toe game written in the Qt/C++ framework.

## Installing
A step by step series  that tell you how to get a execute project.
Get it from GitHub
```
git clone git@github.com:Przemekkkth//tic-tac-toe-qt.git
```
Compile
```
qmake && make
```
